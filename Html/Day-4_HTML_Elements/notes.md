# HTML Element:

-> A tag which has opening and its closing tag along with the content this complete structure called as Element in HTML.

For Ex: <h1>My webpage</h1>
Here

<h1> = Opening tag
</h1> = Closing tag
My Webpage = Content

# What is Void or Empty Elements?

-> The elements which doesn't consist of any kind of content is known as Void or Empty elements.

For Ex: <img/>, <br>, <hr>, <meta>, <input> etc.

# Types of Elements:

-> There are mainly 3 types of elements in HTML: 1. Block-Level Elements 2. Inline-Level Elements 3. Inline-Block Level Elements

1. Block-Level Elements:
   -> The elements which can acquire the entire width of its parent container is known as Block Level Elements.
   -> Block Level elements always starts from a new line.
   -> We can change their height and width for the Block-Level Elements.
   -> For Ex: All Headings Tags, <p>, <div>, <header>, <footer>, etc.

2. Inline-Level Elements:
   -> The elements which can takes the exact width of the content itself is known as Inline-Level Elements.
   -> Inline-Level Elements will come in a same line itself.
   -> We cannot change the height and width for the Inline-Level Elements.
   -> For Ex: <span>, <a>, <button>, <input>, etc.

3. Inline-Block Level Elements:
   -> It is a hybrid combination of Inline and Block Level Elements.
   -> Means it will come in a same line itself as well as we can change their height and width.
   -> For Ex: <img> tags
