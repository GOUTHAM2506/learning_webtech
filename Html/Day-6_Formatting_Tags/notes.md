# Formatting Tags:

-> The formatting tags are used to format the text for better look(visual apperance) and feel.
-> Formatting Tags are Inline-Level Elements.
-> Formatting tags ae divided into 2 tags: 1. Physical Tags 2. Logical Tags

1. Physical Tags:
   -> The Physical Tags are used to provide the just pyhysical apperance for the text.

For Ex: <b>, <i>, <u>, etc.

2. Logical Tags:
   -> The Logical Tags are used to provide the meaning or logic to the text.

For Ex: <strong>, <ins> -> inserted, <em> -> Emphasized.

# Some Types of Formatting Tags:

1. <b>...</b> -> To make text as bold.
2. <u>...</u> -> To provide uderline to the text.
3. <i>...</i> -> It is used to make text slightly tilted(Italic).
4. <sup>...</sup> -> sup stands for SuperScript tag (Above the baseline).
5. <sub>...</sub> -> sub stands for SubScript tag (Below the baseline).
6. <strike>...</strike> -> To provide the strike through to the text (Deprecated = No Longer in use).
7. <del>...<del/> -> It is used to represent the deleted text.
8. <mark>...</mark> -> It is used to highlight the text.
9. <small>...</small> -> To make text little bit small compared to normal text.
10. <big>...</big> -> To make text little bit big compare to normal text.
11. <strong>...</strong> -> Logical Formatting Tag -> It is used make text as strong means bold as well as logical.
12. <ins>...</ins> -> Logical Formatting Tag -> It represents the text inserted in my document (Provides underline).
13. <em>...</em> -> Logical Formatting Tag -> It is used to make text as Italic.
14. <code>...</code> -> To add monospace font to the text.

# Sematic Tags:

-> It is used to add the meaning to HTML tag.
-> Sematic tags are used to tell the information or meaning or to the browser as well as user/programer.

# Types of Sematic Tags:

1. <header> -> Top section (logo,title).
2. <nav> -> Navigation links
3. <main> -> Main content area
4. <section> -> It is used to group the content
5. <article> -> Independent/Self-contained content (headings, paragraphs, along with images)
6. <aside> -> Sidebar / Indirectly related content
7. <footer> -> Bottom section (copyright, links, address)
8. <figure> -> Groups media (image, video, etc.)
9. <figcaption> -> Caption / Description of that media
10. <mark> -> Highlights important text
11. <time> -> Represents date / time
12. <address> -> Contact infromation
13. <details> -> Collapsible content
14. <summary> -> Visible title/heading for <details>

# Why do we need sematic tags in HTML?

-> It improves the SEO (Search Engine Optimization) and makes the code easier to read and mantain
-> Sematic tags helps search engines like Google, understands your website better.
